# Gestión de Proyectos Educativos

Aplicación web desarrollada con React y Vite para gestionar proyectos educativos.

## Funcionalidades

- Visualizar proyectos
- Agregar proyectos
- Buscar proyectos
- Eliminar proyectos

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS
